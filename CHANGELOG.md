## [1.0.0] - 2026-02-12

### Added
- Initial release of @test-company/ui
- Core components: Button, Input, Table
- Theme system with light/dark mode support
- API-driven theme loading
- TypeScript support
- Comprehensive Storybook documentation
- Full Tailwind CSS v4 integration
- Radix UI primitives integration

### Components
- **Button** - Multiple variants (default, secondary, destructive, outline, ghost)
- **Input** - With label, error states, and helper text
- **Table** - Sortable, filterable, with sticky headers
- **ThemeProvider** - Dynamic theming system
